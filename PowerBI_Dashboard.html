Power BI Implementation Guide

Step-by-step instructions to rebuild this dashboard in Microsoft Power BI Desktop

Step 1 — Data Import & Transform
1. Open Power BI Desktop → Get Data → Text/CSV → European_Bank.csv
2. Open Power Query Editor
3. Set column types: CreditScore, Age, Tenure, NumOfProducts, HasCrCard, IsActiveMember, Exited → Whole Number
4. Balance, EstimatedSalary → Decimal Number
5. Close & Apply
Step 2 — DAX Calculated Columns
EngagementScore = [IsActiveMember]*3 + MIN([NumOfProducts],3) + [HasCrCard] + MIN([Tenure]/2,2) EngagementTier = IF([EngagementScore]>=7,"High",IF([EngagementScore]>=4,"Medium","Low")) RetentionStrength = [EngagementScore]*0.4 + ([NumOfProducts]+[HasCrCard]*0.5)*0.6 HighValueDisengaged = IF([Balance]>PERCENTILE([Balance],0.75) && [IsActiveMember]=0, 1, 0)
Step 3 — DAX Measures
Churn Rate = DIVIDE(COUNTROWS(FILTER(Table,[Exited]=1)),COUNTROWS(Table)) HVD Count = COUNTROWS(FILTER(Table,[HighValueDisengaged]=1)) Avg Engagement = AVERAGE(Table[EngagementScore]) Active Member % = DIVIDE(COUNTROWS(FILTER(Table,[IsActiveMember]=1)),COUNTROWS(Table))
Step 4 — Visuals to Build
Page 1 — Overview: Card KPIs, Donut (Exited), Clustered Bar (Geography), Line (AgeGroup vs ChurnRate), Matrix Heatmap (Geography × Gender)

Page 2 — Engagement: Bar (EngagementTier), Gauge (EngagementScore), Scatter (EngagementScore vs Balance, color = Exited)

Page 3 — Products: Bar (NumOfProducts vs ChurnRate), Treemap (ProductIndex), Clustered Bar (Products × EngagementTier)

Page 4 — Risk: Funnel (RetentionTier), Card (HVD Count), Decomposition Tree (Exited by EngagementTier, Geography, NumOfProducts)
Step 5 — Slicers (Filters)
Add these slicers to every page:
• Geography (Dropdown or Tile style)
• Gender (Toggle button)
• EngagementTier (Checkbox list)
• Age (Slider / Between)
• IsActiveMember (Toggle: 0/1)

Tip: Use Sync Slicers (View menu) so filters apply across all pages.
Step 6 — Theme & Branding
View → Themes → Customize:
Primary: #1F3B73 (Dark Navy)
Secondary: #2E6FD8 (Blue)
Accent 1: #E8580A (Orange — alerts)
Accent 2: #0E9B8A (Teal — positive)
Diverging: Red #C0392B → Green #1A9641
Font: Segoe UI throughout
Background: #F0F2F8 (Light slate)
⭐ Power BI Pro Tip: Use the Decomposition Tree visual on the Risk page — set Analyze = "Exited" and Explain By = EngagementTier, Geography, NumOfProducts, Gender. Power BI will automatically identify which combination of factors best explains churn, giving you AI-driven insights without any extra coding.
